  • /help

    Send command list.

  • /avatar [User]

    Display selected user's avatar.

  • /invite

    Send bot invite link.

  • /s

    Play slots game.

  • /binary [Number]

    Convert decimal number to binary.

  • /timer [Time]

    Set timer for selected time.

  • /random [Max]

    Get random number from 0 to [Max].

  • /msgcount [User]

    Get amount of messages sent by selected user.

  • /ban [User]

    Ban user.

  • /kick [User]

    Kick user.

  • /mute [User] [Time] [Reason]

    Mute user.

  • /bc [Content] [Title]

    Send embed message.

  • /hyperlink [Link] [Text]

    Send hyperlink.

  • /giveaway [Time] [Prize]

    Create giveaway.

  • /settings

    Settings list.

  • /unban [ID]

    Unban user.

  • /sblocklinks [True/False]

    Disable/Enable blocking links feature.

  • /scolor [HEX]

    Set embed color.

  • /sautorole [Role]

    Set role that will be automatically added to every user that joins guild.

  • /sjoinmsg [Message] [Channel]

    Set message that will be sent when user join your server. Use
    <user> for user mention and for <guild> guild name.

  • /sleavemsg [Message] [Channel]

    Set message that will be sent when user leave your server. Use
    <user> for user mention and for <guild> guild name.
