Why tBot?

tBot has many functions, like moderation commands and auto deleting links to discord servers. It has also timer that you can set even on 4 days! It pings you when time ends so you don't lose track of time. There is command with which you can set many various settings(ex. embed color). Very advanced broadcast command will help you with making announcements on your discord server!

    Settings

    ADMINISTRATOR permissions required
  • .settings blocklinks [y/n]

    Disable/Enable blocking links to discord servers

    Set embed color

    Set role that will be automatically added to every user that joins guild.

    Set message that will be sent when user join your server.

  • .settings joinmsg channel [channel]

    Set channel where joing messages will be sent.
